# Secure Funds Transfer Application

## Overview
A full-stack web application that allows authenticated users to securely transfer funds between accounts.

The application includes authentication, protected routes, and transaction processing.

## API Endpoints

Authentication

POST /api/auth/login
POST /api/auth/register

Transfers

POST /api/transfers
GET /api/transaction

Account
GET /api/account

## Local Setup

### Backend

1. Clone repo  git clone https://github.com/DaliaMahmoud-sudo/secure-funds-transfer-backend
2. Run  mvn spring-boot:run
 Backend runs on http://localhost:8080
   
---

### Frontend

1. Clone repo  git clone https://github.com/DaliaMahmoud-sudo/secure-funds-transfer-frontend

2. Install dependencies npm install

3. Run project ng serve -o

Frontend runs on http://localhost:4200
